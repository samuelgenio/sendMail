# sendMail
Simple sending email with node

### Requirements
- Node ^12

### Instalation
```bash
npm install
```

## Usage
```bash
npm run dev
```
```bash
curl --location --request POST 'localhost:3333/send' --form 'name="Samuka"' --form 'mail="destino@email.com"' --form 'content="my body is over the ocean"'
```
## Build
```bash
npm run build
npm run start
```
